# DSA-Course

## 🚀 About This Repository

A curated collection of Data Structures and Algorithms (DSA) implementations in C++ designed for: interview prep, competitive coding, and strong portfolio showcase.

- Solid command of arrays, strings, pointers, recursion, sorting, searching, and memory management
- Clean, readable, commented source code in modular folders
- Focus on performance, edge case handling, and practical problem solving

## 🧠 Skills Highlight

- C++ (STL, pointers, recursion, OOP fundamentals)
- Core DSA: arrays, linked lists, stacks, queues, trees, graphs, heaps, hashing
- Algorithms: binary search, merge sort, quick sort, recursion, backtracking, dynamic programming
- Problem-solving mindset with complex-case thinking and optimization awareness

## 📂 Repository Structure

- `GCD.cpp`, `Modular_Exponentiation.cpp`, `Segmented_Sieve.cpp` etc. (math & number theory)
- `Binary Search in 2D array/` (multi-dimensional search patterns)
- `Macros/` (macros, inline functions, default args, global variables)
- `Pointers in C++/` (pointer practice, memory references)
- `Recursion/` (recursive searching/sorting, string reversal, palindrome)
- `Reference Var & Static Vs Dynamic Memory/` (dynamic arrays, jagged arrays)
- `Stack and Queue/` (stack/queue implementations and applications)

## ⭐ Featured Code Samples

### `Recursion/Day5 Merge Sort/mergeSort.cpp`
- Efficient `O(n log n)` divide-and-conquer sort
- Showcases recursion and in-place merging with full comments

### `Segmented_Sieve.cpp`
- Optimized prime number generation for large ranges
- Demonstrates memory/time tradeoffs and competitive problem patterns

### `Pointers in C++/Part_3.cpp`
- Safe pointer arithmetic and reference use
- Includes best practices for avoiding memory leaks and undefined behavior

## 🛠️ How to Run

1. Clone this repo
2. Open in VS Code or your preferred C++ IDE
3. Compile a file:

```bash
g++ -std=c++17 -O2 GCD.cpp -o GCD
./GCD
```

4. For folder-based code, compile each target file individually.
